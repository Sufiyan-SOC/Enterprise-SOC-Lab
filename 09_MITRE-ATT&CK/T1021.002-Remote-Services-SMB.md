# T1021.002 - SMB / Windows Admin Shares

## Technique

SMB / Windows Admin Shares

## Tactic

Lateral Movement

## Lab Scenario

A remote command was executed using Impacket PsExec from the Kali attack workstation to CLIENT1.

## Detection

- Microsoft Sysmon Event ID 1
- SYSTEM-level process creation

## SIEM Detection

Splunk Enterprise detected remote command execution by monitoring Sysmon process creation events.

## Investigation Data

- Endpoint
- Parent Process
- Executed Command
- User Context
- Timeline

## Mitigation

- Restrict administrative shares.
- Limit remote administrative privileges.
- Monitor SYSTEM-level process execution.
- Correlate authentication with process creation.

## Detection Outcome

The simulated Impacket PsExec activity was successfully detected, investigated, documented, and mapped to MITRE ATT&CK.
