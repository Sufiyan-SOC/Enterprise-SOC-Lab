# T1059.001 - PowerShell

## Technique
PowerShell

## Tactic
Execution

## Lab Scenario
Simulated PowerShell execution activity on Windows endpoint.

## Detection

- Sysmon Event ID 1 (Process Creation)
- Windows PowerShell Logs

## SIEM Detection

Splunk used to monitor PowerShell execution and identify suspicious command activity.

## Investigation Data

- Executed Command
- User Account
- Process Name
- Parent Process
- Timestamp

## Mitigation

- PowerShell logging enabled
- Constrained Language Mode
- Monitor suspicious commands
- Apply least privilege access
