# T1021.001 - Remote Desktop Protocol

## Technique

Remote Desktop Protocol (RDP)

## Tactic

Lateral Movement

## Lab Scenario

A controlled Remote Desktop session was established from the Kali Linux attack workstation to a Windows 11 Enterprise endpoint.

## Detection

- Windows Security Event ID 4624
- Logon Type 10

## SIEM Detection

Splunk Enterprise detected successful Remote Desktop authentication and correlated user, source IP, and destination endpoint information.

## Investigation Data

- Source IP
- Destination Endpoint
- User Account
- Authentication Timeline
- Logon Type

## Mitigation

- Restrict RDP exposure.
- Implement Multi-Factor Authentication.
- Monitor successful RDP sessions.
- Review remote administrative access regularly.

## Detection Outcome

The simulated RDP session was successfully detected, investigated, documented, and mapped to MITRE ATT&CK within the Enterprise SOC Lab.
