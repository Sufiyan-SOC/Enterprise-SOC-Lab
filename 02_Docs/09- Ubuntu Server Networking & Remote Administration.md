# Day 09 – Ubuntu Server Networking & Remote Administration

## Overview

Configured the Ubuntu Server to support centralized log management by establishing reliable network connectivity, enabling secure remote administration, and preparing the operating system for future Splunk Enterprise deployment.

---

## Objectives

- Configure Ubuntu Server networking
- Enable secure remote administration using SSH
- Install essential Linux administration utilities
- Prepare the server environment for SIEM deployment

---

## Tasks Completed

- Configured persistent network settings using Netplan
- Added a Host-Only adapter to enable communication between the Host machine and Ubuntu Server
- Installed and configured the OpenSSH Server
- Verified remote SSH connectivity from the Windows Host
- Updated installed packages to the latest available versions
- Installed commonly used Linux administration tools
- Created the SOC-Lab working directory for project organization

---

## Skills Demonstrated

- Ubuntu Server Administration
- Linux Network Configuration
- Netplan Configuration
- OpenSSH Configuration
- Linux Package Management
- Secure Remote Administration

---

## Result

The Ubuntu Server was successfully configured for stable network communication and secure remote administration, providing a reliable foundation for the upcoming Splunk Enterprise installation.
