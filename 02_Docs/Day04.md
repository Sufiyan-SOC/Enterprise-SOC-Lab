# Day 4 - Windows 11 Client Setup

## Objective

Deploy a Windows 11 client and connect it to the Active Directory domain.

## Tasks Performed

- Created a new Windows 11 virtual machine in VirtualBox.
- Installed Windows 11 operating system.
- Configured networking.
- Joined the client computer to the `corp.local` Active Directory domain.
- Restarted the machine.
- Logged in using a domain user account.

## Result

The Windows 11 client successfully joined the domain and authenticated against the Domain Controller.

## Skills Learned

- Windows Client Deployment
- Active Directory Domain Join
- Domain Authentication
- Basic Enterprise Network Configuration
