
Day 1 - Enterprise SOC Lab

Project Name

Enterprise SOC Lab - Attack Detection, Threat Hunting and Incident Response using Wazuh, Splunk, Active Directory and Sysmon

Objective

The objective of this project is to build a real-world Enterprise Security Operations Center (SOC) Lab to gain hands-on experience in security monitoring, log analysis, threat hunting, incident detection, and incident response.

Tools Used

- VirtualBox
- Windows Server 2022
- Windows 11
- Ubuntu Server 22.04 LTS
- Kali Linux
- Active Directory
- Sysmon
- Wazuh
- Splunk Enterprise
- TheHive
- Shuffle SOAR
- VirusTotal API
- GitHub

System Configuration

- Operating System: Windows 11
- Processor: AMD Ryzen 5 5600H
- RAM: 8 GB
- Storage: 500 GB SSD
- Virtualization Software: VirtualBox

Expected Outcome

- Build a complete Enterprise SOC Lab.
- Detect and investigate cyber attacks.
- Perform threat hunting using Splunk and Wazuh.
- Document incidents professionally.
- Publish the project on GitHub as a job portfolio.
