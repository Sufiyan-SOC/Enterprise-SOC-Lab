# Detection Name

DNS Query Monitoring

---

## Detection ID

DR-008

---

## Detection Purpose

Detect DNS queries generated by Windows processes to identify suspicious domain resolution, malware communication, and command-and-control activity.

---

## Detection Type

Behavior-based

---

## Detection Status

Production Ready

---

## MITRE ATT&CK

T1071.004 – DNS

---

## Data Source

Microsoft Sysmon

Event ID 22 – DNS Query

---

## Detection Logic

Monitor DNS queries performed by Windows processes. Investigate unusual domains, unexpected processes performing DNS lookups, and repeated queries to external domains.

---

## Splunk Detection Query

```spl
index=* source="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational" EventID=22
| table _time Image QueryName QueryResults User
```

---

## Severity

Medium

---

## False Positives

* Web browsers
* Windows Update
* Enterprise applications
* DNS cache refresh

---

## Triage Steps

1. Review querying process.
2. Validate queried domain.
3. Review resolved IP.
4. Verify user context.
5. Correlate with network connections.

---

## Recommended Analyst Actions

* Validate domain reputation.
* Investigate unusual DNS activity.
* Correlate with Sysmon Event ID 3.
* Escalate confirmed suspicious communication.
