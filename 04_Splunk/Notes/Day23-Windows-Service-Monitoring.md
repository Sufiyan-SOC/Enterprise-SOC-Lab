# Windows Service Monitoring

## Purpose

Windows Service monitoring helps detect persistence techniques where attackers install malicious services to maintain long-term access.

## Windows Event

Event ID 7045 – Service Installed

## Important Information

* Service Name
* Service File Name
* Service Account
* Start Type

## SOC Investigation Focus

* Unknown services
* Auto-start services
* Executables outside Windows or Program Files
* Fake Microsoft service names
* Persistence mechanisms
