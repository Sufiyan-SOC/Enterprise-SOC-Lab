# Windows Authentication Monitoring

## Event ID 4624

Successful Login

## Event ID 4625

Failed Login

## Investigation Steps

- Identify the affected user
- Count failed logins
- Build authentication timeline
- Check for successful login after failures
- Determine whether activity is legitimate or suspicious

## SOC Use Cases

- Password Guessing
- Brute Force Attack
- Account Lockout Investigation
- User Login Monitoring
