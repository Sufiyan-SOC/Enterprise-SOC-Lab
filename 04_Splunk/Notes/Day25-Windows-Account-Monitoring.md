# Windows Account Management Monitoring

## Purpose

Monitor account creation, deletion, password changes, and privilege changes to identify unauthorized administrative activity.

## Windows Security Events

* 4720 – User Account Created
* 4722 – User Enabled
* 4723 – Password Changed
* 4724 – Password Reset
* 4725 – User Disabled
* 4726 – User Deleted
* 4732 – User Added to Group

## SOC Investigation Focus

* Unexpected user creation
* Administrator-created accounts
* Privilege escalation
* Insider threats
* Persistence through valid accounts
