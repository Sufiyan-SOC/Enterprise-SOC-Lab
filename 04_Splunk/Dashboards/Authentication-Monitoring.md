# Authentication Monitoring Dashboard

## Purpose

Provides centralized visibility into Windows authentication activity.

---

## Dashboard Panels

### Failed Login Attempts

Displays failed authentication events over time.

---

### Successful Login Attempts

Displays successful authentication events.

---

### Top Target Accounts

Shows the most frequently targeted accounts.

---

### Top Source IP Addresses

Shows systems generating failed authentication attempts.

---

### Recent Failed Authentication Events

Displays recent authentication failures.

---

### Authentication Timeline

Provides an investigation timeline for authentication events.
